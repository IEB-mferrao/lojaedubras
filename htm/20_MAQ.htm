Unidade 01
Depilação de sobrancelhas. Remoção da maquiagem. Apresentação de materiais. Maquiagem para o dia. Maquiagem para a noite. Maquiagem para noivas. Modelagem das sobrancelhas.

Unidade 02
Maquiagem para Quinze Anos. Maquiagem para senhoras. Maquiagem para pele morena. Maquiagem para passarela. Maquiagem para homens. Maquiagem de debutante ou 15 anos. Maquiagem natural para o dia.

Unidade 03
Estruturação do rosto. Maquiagem para fotografia em preto-e-branco. Maquiagem para publicidade.

Unidade 04
Diferentes tipos de maquiagem para o mesmo rosto.

Unidade 05
Maquiagem para louras de olhos claros.

Unidade 06
Maquiagem de noite para passarela. Variação de Maquiagem.

Unidade 07
Maquiagem destacada para a noite.
