ASSISTA TRECHO DE UM VÍDEO

Unidade 01 - Descrição do Computador

Descrição de um Computador. Desmontagem de um CPU e explicação de suas diferentes unidades: disco rígido, leitora de CD, disqueteira. Fitas de transferência de dados IDE. Placa Mãe: descrição de suas unidades.

Unidade 02 - Conceitos Teóricos

Fontes de alimentação: tensão de corrente de saída, fontes ATX e AT. Arquitetura e Buses. Memórias: volátil e não volátil. Encapsulados SIMM, DIMM, DDR. Fontes AT e ATX. Provas de possíveis falhas e conserto das mesmas. Monitores Standard (possíveis falhas e consertos) e de cristal líquido. Disco rígido, desmontagem. Leitora de CD, disqueteira, teclado, mouse. Mouse óptico e de bolinha (desmontagem e limpeza).

Unidade 03 - Instalação e Configuração de Diferentes Dispositivos

Gravador de CD. Disqueteira. Disco rígido. Impressoras: de ponto, tinta, laser. Desmontagem e instalação scanner. Desmontagem e instalação de Windows XP. Instalação de Windows XP de 64 bits. Fdisk. Scandisk. Scandisk do Windows XP. Desfragmentação. Instalação de controladores de hardware. Instalação de antivírus Norton Ghost. Instalação do Linux. Controladores de recuperação do Windows XP. Truques do Windows XP. Possíveis falhas do hardware ou software. Aplicações para utilizar com problemas de software. Back-up. Carga de cartuchos de impressora.

Unidade 04 - Redes

Funcionamento de uma rede: Intranet, Internet, Extranet, Lan, Wan, Man. Protocolos. Portos. Elementos de uma rede: (Utp, RJ45, Patcheras, Patchcod, Firewall, DHCP, DNS, Servidor, Servidor Proxy, Servidor Web. Servidores por software e hardware. Topologias de redes. ADSL, DSL, Dial-Up. Instalação da rede. Montagem de cabo com terminais RJ45. Protocolos: http, ftp, Smtp, Pop. Protocolo de comunicação TCP/IP (protocolo de Internet). IP's. Formato do pacote TCP/IP.

Unidade 05 – Instalações de Redes

Instalação de interfases de rede (NIC): através de software e hardware. Instalação de uma intranet entre dois PC’s. Amostra de ferramentas necessárias. Dispositivos ativos e passivos. Configuração dos PC’s clientes ou host. Instalação de Servidor Win2003 Server. Configuração de serviços: firewall, compartilhar Internet, usuários, turmas, seguridade. Teste da rede: Ping, NetStat, para testar defeitos de conectividade na rede. Configuração de ADSL. Instalação de Servidor Web (Apache). Instalação de um serviço proxy de acesso compartilhado à Internet. Instalação de firewall. Escritório remoto. Instalação e configuração de wi-fi.
